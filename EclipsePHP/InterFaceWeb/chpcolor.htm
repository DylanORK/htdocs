Champ de saisie de couleurs
Ce script permet de créer des champs de saisie avec indicateur de couleurs avec une seule ligne de code du type :
<SCRIPT language=javascript>TJS_PrintChoixColor("fond","red");</SCRIPT>
Cette fonction crée un champ texte appelé fond avec la couleur prédéfinie red.
Vous pouvez créer autant de champs que vous voulez dans une page.
